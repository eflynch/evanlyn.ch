learn how to edit in the default 'vim' mode

  • so far we have been navigating this site in 'vim' mode. if that is familiar to you, then this should be easy
  • practice basic editing with 'i' and 'Escape'
  • practice basic editing with 'c' and 'Escape'
  • practice deleting items/trees with 'shift+d'
  • practice moving items around with shift+(hjkl)
  • practicing creating new items with 'o' and 'shift+o'
  • 'u' will undo the last thing you did
  • 'r' will redo the last thing you undid
  • shift+E embeds the contents of the current item as a link to content
  • shift+F embeds the contents of the current item as an external link
  • shift+W copies the entire contents of the application in JSON to the clipboard
