learn how to edit in 'std' mode

  • you can't currently do as many things in 'std' mode
  • you can always edit the selected item and there is no special navigation mode
  • instead you should navigate with the arrow keys
  • shift+spacebar collapses/expands
  • alt+> sends you into the selected sublist
  • alt+< brings you out of a sublist
  • you can also move items around with shift+arrow
